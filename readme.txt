This project, based on the SOFEA architecture, is a pokedex comprising :
- A view that displays a list of pokemon as tiles (with its id, name and an image).
- A detailed view of a pokemon (with its id, name, image and characteristics of the pokemon).
- A search bar for a pokemon (by id and name).

############################################################################################################

Note : To launch the pokedex, simply open the index.html file in a web browser.

############################################################################################################

Project carried out as part of a web architecture course at Polytech Montpellier.
